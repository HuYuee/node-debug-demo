# node-debug-demo
来测试方便node调试开发的工具

## 方式
- 使用node自身debug参数调试
- 使用node-inspector插件
- 使用vscode编译器

## 总结
最后发现使用vscode实在是太方便了，完爆所有其他插件或者库，用ide还是最方便的！大力推荐
